# 📝 Today I Learned

Today I Learned(TIL)은 그날그날 새롭게 배운 내용을 기록하는 학습 방법입니다.

꾸준한 기록을 통해 자신의 성장을 눈으로 확인할 수 있고, 나중에 필요할 때 쉽게 찾아볼 수 있다는 장점이 있습니다.

블로그를 통해서 작성해도 좋지만, 커밋이 기록되는 깃이 더 효과적인거 같습니다.
